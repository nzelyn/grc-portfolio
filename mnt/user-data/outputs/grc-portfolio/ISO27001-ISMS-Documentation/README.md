# ISO 27001 ISMS Documentation — NexaCloud Solutions

**Standard:** ISO/IEC 27001:2022  
**Certification Status:** Pursuing Initial Certification (Stage 2 Audit: Q1 2025)  
**ISMS Scope:** NexaCloud SIEM Platform and supporting AWS infrastructure  
**Document Owner:** CISO  
**Last Reviewed:** September 2024

---

## Project Summary

This project contains a complete ISO 27001:2022 Information Security Management System (ISMS) documentation set for NexaCloud Solutions. The documentation covers ISMS scoping, all 93 Annex A controls in the Statement of Applicability, a full risk assessment with treatment plan, asset inventory, internal audit findings, management review minutes, and corrective action records.
