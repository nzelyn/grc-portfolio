# Security Policy Repository — NexaCloud Solutions

**Frameworks:** SOC 2 Type II, ISO 27001:2022  
**Policy Owner:** CISO  
**Review Cycle:** Annual (or upon material change)  
**Last Reviewed:** September 2024

---

## Project Summary

This repository contains five enterprise-grade information security policies for NexaCloud Solutions. Each policy follows a consistent format, includes a document control table, version history, and approval records. Policies are aligned to SOC 2 TSC requirements and ISO 27001:2022 Annex A controls.

## Policy Approval Workflow

```
Draft by Policy Owner
        |
        v
Technical Review (Engineering + IT Operations)
        |
        v
Legal Review (if data privacy or regulatory content)
        |
        v
Security Committee Review
        |
        v
CISO Approval
        |
        v
CEO Sign-Off (for Tier 1 policies)
        |
        v
Publish to Policy Portal (Confluence)
        |
        v
All-Staff Communication + Acknowledgement
        |
        v
Annual Review Cycle
```
