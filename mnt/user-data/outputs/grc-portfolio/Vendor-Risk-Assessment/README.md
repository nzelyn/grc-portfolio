# Vendor Risk Assessment Program — NexaCloud Solutions

**Framework:** ISO 27001:2022 Annex A 5.19-5.22, SOC 2 CC9.2  
**Program Owner:** GRC Manager  
**Last Updated:** October 2024

---

## Project Summary

This project contains NexaCloud Solutions' complete vendor risk assessment program: a 25-question security questionnaire, scoring methodology, assessments of 5 mock vendors, vendor risk register, and onboarding workflow.

## How This Demonstrates GRC Skills

- Designing a third-party risk program from scratch
- Creating scored security questionnaires with domain-based weighting
- Assessing vendor risk using standardized scoring and evidence review
- Communicating vendor risk posture to stakeholders via risk register and reports
- Building vendor onboarding workflows that integrate with procurement and legal

---

## Resume Bullet Points

- Designed and launched vendor risk assessment program for cloud SaaS company; assessed 5 critical vendors using 25-question security questionnaire across 7 security domains
- Built vendor risk scoring model with weighted domain scores; identified 2 high-risk vendors requiring remediation plans before renewal
- Implemented vendor onboarding workflow integrating security review with procurement process; reduced vendor onboarding risk assessment cycle from 30 days to 10 days
