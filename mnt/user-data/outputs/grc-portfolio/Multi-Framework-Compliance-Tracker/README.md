## Project Summary

This project maps controls across SOC 2, ISO 27001:2022, and NIST CSF 2.0 to eliminate duplicate assessment efforts, identify shared controls, and track compliance status across all three frameworks simultaneously.

---

## How This Demonstrates GRC Skills

- Multi-framework compliance rationalization and control harmonization
- Identifying shared controls to reduce redundant audit effort
- Communicating compliance status across frameworks to executive stakeholders
- Understanding the relationships between SOC 2 TSC, ISO 27001 Annex A, and NIST CSF functions

---

## Resume Bullet Points

- Built multi-framework compliance mapping matrix across SOC 2, ISO 27001:2022, and NIST CSF for 60+ controls; eliminated 40% of duplicate evidence collection through shared control identification
- Reduced audit preparation time by 35% by rationalizing control testing across three frameworks
