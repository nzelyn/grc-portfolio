## Instructions

Complete all questions. Attach supporting evidence where indicated (SOC 2 report, ISO 27001 certificate, penetration test summary, etc.). Return to: grc@nexacloud.io within 10 business days.

**Scoring:** Each question is scored 0-3 based on response quality.  
- **3:** Control fully implemented with documented evidence  
- **2:** Control partially implemented or in progress  
- **1:** Control planned but not yet implemented  
- **0:** Control not in place

**Risk Tiers:**  
- Score 85-100%: Low Risk
